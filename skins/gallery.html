This page reads jsondata/gallery.json, which is generated from entries in weewx.conf — there aren't any yet (or the file hasn't been generated).

How to add one (weewx.conf + skin.conf)

Add numbered gallery_item<N>_* keys under this skin's [[[Extras]]] in weewx.conf:

[StdReport]
    [[CommunityWeather]]          <- your skin's stanza name, not literally this
        skin = community_weather
        [[[Extras]]]
            gallery_item1_title    = Live Radar
            gallery_item1_category = map
            gallery_item1_src      = https://example.com/embed/radar
            gallery_item1_aspect   = 4:3
            gallery_item1_notes    = Refreshes every 5 minutes

            gallery_item2_title    = Garden Webcam
            gallery_item2_category = webcam
            gallery_item2_src      = https://example.com/embed/cam1
            gallery_item2_aspect   = 16:9

category is one of map / webcam / timelapse — anything else (or left blank) lands under "Other". aspect and notes are optional (aspect defaults to 16:9).

Then make sure gallery.json.tmpl (shipped alongside this page) is being generated by skin.conf's [CheetahGenerator] section — mirror whatever stanza this skin already uses for its other *_json.tmpl files (e.g. charts_json.tmpl) so it lands at jsondata/gallery.json. See gallery.json.tmpl's own header comment for the full, exact instructions and a worked example.
