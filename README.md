# weewx-sael 

# SǢL means Season in Old English

Features include:

    Modern, responsive card-based layoutd
    Real-time streaming updates (no MQTT broker required)
    Extensive charting system with full customized control on observations, historical timescale, grouping and more using EChart charting libraries
    Light and dark mode support (manual toggle or auto sunrise and sunset switching between light and dark modes)
    Real-time switching between various unit groups (UK, US, Metric, Scandinavian, Canada, Aviation, Beaufort)
    Forecast data (both at the station location and for any other location when you are on the move)
    Almanac (sun/moon) data
    Weather records -- day, week, month, year, all time
    All information updates automatically, so page refreshes are not necessary
